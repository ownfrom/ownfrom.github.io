安装git

  • 下载git安装包直接安装就可以

配置GitHub

  • 创建GitHub窗户，设置用户名，邮箱
  • 创建仓库，仓库名为：用户名.github.io

配置SSH key

  • 配置git的用户名和密码，要与GitHub一致，在桌面右击=>Git Bash Here输入：
    • git config --global user.name "GitHub用户名"，回车
    • git config --global user.email "GitHub邮箱"，回车
  • 生成SSH key
    • ssh-keygen -t rsa -C "邮箱"
    • 记事本打开 C:\Users\WK-PC\.ssh\id_rsa.pub，ctrl+A复制
    • 打开GitHub，点击头像=>setting=>SSH and GPG keys=>New SSH key，title随意，在key中粘贴，点击add SSH key
    • 输入ssh -T git@github.com，回车
    • 如果提示 Are you sure you want to continue connecting (yes/no)? ，输入yes，然后会看到：Hi liuxianan! You've successfully authenticated, but GitHub does not provide shell access. 说明已经配置成功

安装node.js

  • 官网下载node.js,下载LT版本，LTS为长期支持版，Current为当前最新版
  • 在Git Bash中分别输入node -v与npm -v回车查看版本号

配置hexo

  • 安装hexo

    • 输入npm install -g hexo ，回车

  • 初始化

    • 在随意创建一个文件夹，我直接用的用户名：F:\ownfrom,关闭gie，在文件夹中重新打开git

    • 输入hexo init,在文件夹中会出现多个文件夹和文件

      • source：用来存放头像图片，博客文章等
      • themes：用来存放主题
      • _config.yml：配置文件

    • 配置 _config.yml

      • 将最后的 deploy:改为

      • 修改 url 和 root

        • url: https://用户名.github.io/用户名.github.io/
        • root: /用户名.github.io/

    • 输入hexo clean ，清楚缓存

    • 输入hexo g，上传到本地服务器

    • 输入hexo s，启动本地服务器；

      • 浏览器中输入：localhost4000就可以访问，默认端口是4000；ctrl+c关闭服务

    • 输入hexo d，上传到GitHub

      • 在浏览器输入 https://用户名.github.io/用户名.github.io/ 就可以访问个人博客了

    • 如果提示Warning: Permanently added the RSA host key for IP address '140.82.114.4' to the list of known hosts.，打开系统的host文件C:\Windows\System32\drivers\etc\hosts，在最后添加 提示的ip地址 github.com，如果还提示不同的 id，就再添加

    • 如果网页一直加载，打开控制台提示错误，可能是 url 和 root错误
